# 장쌤 AI 블로그 기획 워크시트 사용 가이드

## 1) 이 구성은 어떻게 쓰나요?
- 학생은 `index.html` 화면에서 워크시트를 사용합니다.
- 강사는 필요할 때만 `server.js`를 실행해서 실제 AI 응답을 연결합니다.
- 학생이 무료 ChatGPT 사용자여도 상관없습니다. API는 학생이 아니라 **강사 컴퓨터의 서버**에서만 사용합니다.

## 2) 무료 모드로 쓰는 방법
가장 쉬운 방식입니다.
1. `index.html`만 브라우저에서 엽니다.
2. 상단 `실제 AI 호출 사용` 체크를 끈 상태로 둡니다.
3. 그러면 템플릿형 로컬 결과로도 전체 수업 흐름을 진행할 수 있습니다.

## 3) 실제 AI 연결 모드
### 준비
1. Node.js 설치
2. 이 폴더에서 터미널 실행
3. 아래 순서대로 실행

```bash
npm install
cp .env.example .env
```

4. `.env` 파일에 본인 OpenAI API 키 입력

```env
OPENAI_API_KEY=여기에_API_키_입력
PORT=3000
```

5. 서버 실행

```bash
npm start
```

6. 브라우저에서 아래 주소 열기

```text
http://localhost:3000
```

7. 워크시트 상단에서 `실제 AI 호출 사용` 체크
8. `연결 테스트` 버튼 클릭

## 4) 수업에서 가장 쉬운 운영 방식
- 학생 개인 컴퓨터마다 API를 넣지 않습니다.
- 강사 노트북 1대에서만 서버를 켭니다.
- 강사 시연용으로만 실제 AI를 사용합니다.
- 학생 실습은 대부분 무료 모드로 진행합니다.
- 필요한 학생만 강사 컴퓨터에서 1:1로 실제 결과를 보여줍니다.

## 5) 이미지 생성
- `이미지 실제 생성` 버튼은 서버가 켜져 있고 API 키가 있어야 동작합니다.
- 서버가 없으면 이미지 프롬프트만 생성됩니다.

## 6) 자주 생기는 문제
### 연결 테스트 실패
- `.env`에 API 키가 없는지 확인
- 서버가 실행 중인지 확인
- `http://localhost:3000`으로 접속했는지 확인

### 학생이 파일만 열었는데 버튼이 안 되는 경우
- `file://`로 열면 일부 기능이 제한될 수 있습니다.
- 가능하면 서버를 켜고 `http://localhost:3000`으로 접속하세요.

## 7) 추천 운영 팁
- 첫 수업: 무료 모드로 전체 흐름 익히기
- 둘째 수업: 강사가 실제 AI 연결 시연
- 셋째 수업: 일부 학생만 실제 AI 결과 비교
